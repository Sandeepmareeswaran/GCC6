# GCC6 🛒

A full-stack web application serving as a productive integration store,
enabling users to discover, manage, and integrate productivity tools
and services from one unified platform.

## 📌 About
GCC6 is a web-based integration marketplace that connects multiple
productivity tools, allowing users and admins to manage services,
track usage, and configure integrations — all from a single portal.

## 🛠️ Tech Stack
- **Frontend:** React.js + Tailwind CSS
- **Database:** Firebase
- **Auth:** JWT Authentication
- **Deployment:** VPS

## ✨ Features
- Integration marketplace UI
- User & admin dashboard
- Service management and configuration
- Authentication and role-based access
- Responsive design

## 🚀 Getting Started

### Installation
```bash
git clone https://github.com/Sandeepmareeswaran/GCC6.git
cd GCC6
npm install
npm run dev
```

### Environment Variables
Create a `.env` file:
```env
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_jwt_secret
```
